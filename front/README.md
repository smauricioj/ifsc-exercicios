# Exercícios de Front-end I

Exemplos de solução dos exercícios das aulas de Front-end I. A estrutura desse repositório
é:

~~~
.
├── audio
├── img
│   ├── icon
│   └── logos
├── index.html
└── pages
    └── exercicio_n.html
~~~

O index.html é uma página central que possui links individuais para todos os exerícios. Os diretórios organizam os recursos presentes, entre páginas HTML (/pages), imagens (/img) e arquivos de som (/audio).
